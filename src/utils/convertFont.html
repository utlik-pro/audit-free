Конвертер шрифтов для jsPDF

Используйте этот инструмент: https://rawgit.com/MrRio/jsPDF/master/fontconverter/fontconverter.html

Инструкция:

  1. Скачайте шрифт Roboto: Google Fonts - Roboto
  2. Откройте конвертер по ссылке выше
  3. Загрузите TTF файл Roboto-Regular.ttf
  4. Скачайте сгенерированный .js файл
  5. Скопируйте содержимое в src/utils/robotoFont.ts

Альтернатива: Используйте транслитерацию (текущее fallback решение)
